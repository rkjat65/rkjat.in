This template shows you how to add different types of content to your portfolio projects.

1. Embed Streamlit App

If your app is hosted on Streamlit Cloud:

<iframe 
  src="https://your-app.streamlit.app/?embedded=true" 
  width="100%" 
  height="700"
  style="border: none; border-radius: 12px;">
</iframe>

Note: Add ?embedded=true to hide Streamlit's header/footer.

2. Embed Dash/Flask App

If hosted on Render, Railway, or Heroku:

<iframe 
  src="https://your-dash-app.onrender.com" 
  width="100%" 
  height="700"
  style="border: none; border-radius: 12px;">
</iframe>

3. Embed Jupyter Notebook

Convert notebook to HTML and embed:

# In terminal:
jupyter nbconvert --to html your_notebook.ipynb

# Then embed:
<iframe 
  src="../assets/notebooks/your_notebook.html" 
  width="100%" 
  height="800"
  style="border: none;">
</iframe>

Or use nbviewer:

<iframe 
  src="https://nbviewer.org/github/username/repo/blob/main/notebook.ipynb" 
  width="100%" 
  height="800">
</iframe>

4. Embed PDF Report

Place PDF in assets/pdf/ folder:

<iframe

  width="100%" 
  height="600"
  style="border: 1px solid var(--border); border-radius: 12px;">
</iframe>

6. Embed Plotly Charts (No Server Needed)

Export chart from Python, then embed:

# In Python/Jupyter:
import plotly.express as px

fig = px.line(df, x='date', y='value', title='My Chart')
fig.write_html('chart.html', include_plotlyjs='cdn')

# Then in your HTML page:
<iframe 
  src="../assets/charts/chart.html" 
  width="100%" 
  height="500"
  style="border: none;">
</iframe>
